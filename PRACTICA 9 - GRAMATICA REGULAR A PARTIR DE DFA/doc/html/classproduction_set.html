Descripción detallada

Implementación de un conjunto de producciones.

Esta clase implementa, de forma análoga a la clase estado para DFAs/NFAs, un conjunto de producciones productions_ para un determinado símbolo no terminal nonTerminal_

Implementa métodos para acceder/modificar el no terminal, acceder y añadir producciones y la sobrecarga de operadores necesaria al tener que implementar un set<productionSet> en la clase GR.
